SublimeLinter-shellcheck
-------------------------------
This linter plugin for SublimeLinter provides an interface to shellcheck.

INSTALLATION
------------
Before this plugin will activate, you *must* follow the installation
instructions here:

https://github.com/NotSqrt/SublimeLinter-shellcheck

*** WARNING : `shellcheck` has to be compiled from its Haskell sources.
There is no Windows installer, and no Homebrew or MacPorts version available for Mac OS X. ***
